To-do lists help users to keep track of tasks they have to do,

The purpose of the app is to keep track of tasks, allowing the user
to view, add and delete task as they choice, the user inputs the task
into the text box at the top of the screen, clicks add to task, and
the task cna be viewed below along with the date it was added.

To delete the task, dragging the task to the left, will show a delete button
allowing the user to remove tasks they have already completed.
